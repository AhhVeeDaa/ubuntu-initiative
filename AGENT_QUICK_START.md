## üö® TROUBLESHOOTING

### Issue: "Configuration needed" in dashboard

**Fix**:
1. Check Vercel environment variables
2. Verify all 3 variables are set correctly
3. Redeploy the app
4. Clear browser cache and refresh

### Issue: Agents not running on schedule

**Fix**:
1. Verify cron job exists in Vercel Dashboard
2. Check Vercel function logs for errors
3. Verify CRON_SECRET matches in code and env
4. Test manual trigger first

### Issue: Database connection errors

**Fix**:
1. Verify SUPABASE_SERVICE_ROLE_KEY (not SERVICE_KEY!)
2. Check key has not expired
3. Test connection with curl
4. Review Supabase logs

### Issue: Gemini API errors

**Fix**:
1. Check API key is valid
2. Verify quota remaining at https://makersuite.google.com
3. Test with simple prompt
4. Check rate limiting
